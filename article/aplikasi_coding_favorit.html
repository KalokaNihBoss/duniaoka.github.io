Aplikasi coding kesukaan aku

Hai kawan! Apakah kalian tau coding? Coding itu kalo menurut aku membuat aplikasi, game, atau website dengan kode-kode. Tapi kalo mau ngoding harus pake aplikasi seperti apa sih? Kalian bisa menggunakan aplikasi

  1. Notepad++
  2. Sublime Text
  3. Visual Studio Code
Kalian milih yang mana, kalo aku milih Visual Studio Code karena sangat banyak extensionnya. Extension itu pembantu ngoding kalian, walau beberapa extension kadang tidak terlalu membantu dalam ngoding.

Kelebihan VS Code menurut aku :

  • Banyak Extensions
  • Banyak Fitur
  • Gratis
