Tentang Saya

Namaku Kaloka, aku mulai belajar membuat website umur 10 tahun. Aku awal-awal ngebuat website tentang game, tetapi proyek itu aku sudahkan karena aku sudah menemukan cara mendesain dan membuat website dengan lebih cepat, seperti, bootstrap, tailwindcss untuk mempercepat membuat dan mendesain website, ilmuku makin bertambah tiap harinya dan aku rasa website game ku kurang bagus, jadi itu alasan aku udahin ngebuat website gamenya.

Nah adikku juga pengen ikut-ikutan membuat website, tapi hebatnya dia membuatnya di umur 8 tahun, saat itu adikku masih belajar membuat website tanpa css dan javascript tapi makin hari dia makin bisa, tetapi dia udahan ngebuat websitenya karena bosen, padahal DOM, css grid, css layouting, bootstrap, javascript belum dipelajari
